Readme ShoppingList

Function of Git:
	Initialize local repository:
	1. Team -> Remote -> clone
	2. Clone from github.com !!!SET ON FEATURE BRANCH!!!
	
	If local repository is already created:
	1. Team -> Remote -> pull from upstream
	After changes are made:
	1. Team -> commit (Optional commit message)
	2. Team -> remote -> push to upstream

!!!Commits only in feature branch!!!
